        Search for the item "pipebomb" in the sql database and...well.. delete it everywhere you find it~ (a LOT of work^^)

         

        or

         

        replace them by some item .. for example a coke :D (nice trolling ^^)

         

        sql example for that case would be

        update character_data set Inventory = replace(Inventory, 'PipeBomb', 'ItemSodaCoke') where instr(Inventory, 'PipeBomb') > 0;

         

        and repeat that for every field/table the bomb could be (inventory, backpack, inventory of objects)

        like that:

        update character_data set Backpack = replace(Backpack, 'PipeBomb', 'ItemSodaCoke') where instr(Backpack, 'PipeBomb') > 0;

        update object_data set Inventory = replace(Inventory, 'PipeBomb', 'ItemSodaCoke') where instr(Inventory, 'PipeBomb') > 0;

         

        this should do it~

        Have removed satchel charges on our server due to players deliberately blowing up buildings to cause desync!!!

         

        You need to replace the item in the database with something that players are not going to have or you could bork their inventory, especially with a Coke as they're likely to already have one, also applies to storage items. Also, you need to change it in players main inventory. I've chosen to replace them with an AKSD mag as afaik they're not ingame.

        BEGIN

update character_data set Backpack = replace(Backpack, 'PipeBomb', '30Rnd_545x39_AKSD') where instr(Backpack, 'PipeBomb') > 0;

update character_data set Inventory = replace(Inventory, 'PipeBomb', '30Rnd_545x39_AKSD') where instr(Inventory, 'PipeBomb') > 0;

update object_data set Inventory = replace(Inventory, 'PipeBomb', '30Rnd_545x39_AKSD') where instr(Inventory, 'PipeBomb') > 0;

END
