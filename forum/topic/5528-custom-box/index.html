        what halvhjearne already said you have to put a variable on the box

        _this setVariable ["Sarge",1,true];

        _this is the name of the box

        Â 

        for exampleÂ 

        _crate = "MedBox0" createVehicle (position player);
_crate setVariable ["Sarge",1,true];

        and then in your server_cleanup.fsm add this to the check for hackers line

        && (vehicle _x getVariable [""Sarge"",0] != 1)

        That should prevent it from cleaning it up
