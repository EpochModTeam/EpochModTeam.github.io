        Well removing gender check when changing clothes proved to be a rather reliable solution, it only sent me to debug for 2 female skins (probably banned skins) but besides that it works perfectly fine without any issues.

         

        If anyone wants to remove it then open up your player_wearClothes.sqf and find this

        		_currentSex = getText (configFile >> "CfgSurvival" >> "Skins" >> _itemNew >> "sex");
		// Sex of new skin
		_newSex = getText (configFile >> "CfgSurvival" >> "Skins" >> _item >> "sex");

        Change it into this

        		_currentSex = getText (configFile >> "CfgSurvival" >> "Skins" >> _itemNew);
		// Sex of new skin
		_newSex = getText (configFile >> "CfgSurvival" >> "Skins" >> _item);

        Also if you're looking to get rid of one annoying bug when you change clothes with a used mag in your backpack which resets your humanity, eats any guns in backpack and rolls you back on relog just uncomment line #24

         

        Edit: Faulty skins seem to be SurvivorWsequisha_DZ and SurvivorWsequishaD_DZ
