        ok root in the computer world means:

        Â 

        1. The base level aka C:\ or "/" in linux. Â could be used like "the dayz server root directory" which could be "c:\program files (x86)\steam\steamapps\common\arma2OA\"

        2. Admin/Super User - Has root access, aka can modify anything in the root directory (and children) "/" or "C:\"

        Â 

        What you have is a dedicated server. The best way I can think of would be to add the following to your init.sqf and write a script to scrape your log files and output the data to a web interface.

        DZE_DiagFpsFast = false; // reports every 1 Min
DZE_DiagFpsSlow = true; // reports every 5 Min
DZE_DiagVerbose = false; // include objects & players, false = only FPS
