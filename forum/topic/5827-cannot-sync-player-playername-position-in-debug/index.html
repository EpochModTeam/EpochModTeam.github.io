        No idea about the cannot find channel thing, but the position in debug i think it has to do with your mission.sqm

        Search your mission.sqm file for -7239.67. You should find this value there...actually it should be many times there.

        I've changed all those group points to a positive value

         

        position[]={20386.953,0.15662582,-5039.3979};

        position[]={20385.904,0.17103837,-5039.0381};
        position[]={20386.109,0.18125358,-5037.7646};

        ....

        etc..

         

        I get fewer 'in debug' errors this way.

         

        I think these coordinates are the places it takes players when they change clothes or when they join the server, as an initialization starting point.

         

        Load your mission.sqm into arma editor and check where those groups are, then move them all to a place inside the map (an island i guess will do fine).

        For the split second the player gets teleported there by the server (when you switch skins for example) they will be inside the map area, and not debug.
