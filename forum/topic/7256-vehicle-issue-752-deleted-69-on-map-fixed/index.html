        So I've searched through the forums yet can't find an answer to my issue. My server hosted by hfb isn't spawning cars like it has been. I have over 900 spawn locations, 800 max vehicle limit, yet when my server restarted an hour ago it shows it deleted 752 vehicles deleted. Alot were damaged so the basic cleanup script from epoch wiki is working, but how or why was that many deleted? Is there a way to execute an SQL to force spawn these vehicles? Or a way I can make sure the server spawns more? Even with all my custom spawn points and vehicle limit it's never hit higher than 500 unless I used tsw to manually spawn more which takes forever.

        Any help would be appreciated, thanks

         

        EDIT: One of my admins figured it out, it was a problem with Sarge AI that he added in.  He said it was a line that he put in incorrectly that was causing problems with the cleanup and now my server has been spawning vehicles out the ***.
