        Edit the mission.sqm, find class markers and add your own :)

         

        class Markers
    {
        items = 3;
        class Item0
        {
            position[] = {6968.658,324.48145,8345.914};
            name = "center";
            type = "Empty";
        };
        class Item1
        {
            position[] = {1024.82,0,2023.51};
            name = "respawn_east";
            type = "Empty";
            angle = 23.608498;
        };
        class Item2
        {
            position[] = {1024.8502,5.858802,2023.5204};
            name = "respawn_west";
            type = "Empty";
            angle = 23.608498;
        };
    };

         

        Easy way is use the editor, add the marker, give it a name and icon, save the mission and add to your mission.sqm

        If you do it manually remember it's xzy not xyz, height is the middle parameter and can be anything for a marker.

        i make markers from my custom markers.sqf

         

        example:

        _this = createMarker ["187_102x", [18713.1, 10237.1, 0]];
_this setMarkerShape "ELLIPSE";
_this setMarkerColor "ColorBlue";
_this  setMarkerAlpha 0.9;
_this setMarkerSize [80, 80];

        i run it from my custom pbo but it can be run from init.sqf
