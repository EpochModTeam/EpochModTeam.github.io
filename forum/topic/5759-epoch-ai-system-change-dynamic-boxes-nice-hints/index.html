        Hello All, I've been using the AI Mission system and I wanted to share with you some of the modifications I did, maybe you will like them on your server as well.
        This is for people that have some experience with scripting already and can modify the code on their own.
         
        New Hints (with pictures) like Wasteland:
        My mission structure is slightly modified, I just have a random number generated that sets a case for a mission, that sets the mission subtitle, picture, etc etc... and initiates the selected mission. Once the mission is complete, it runs the ending mission complete hint.

        Dynamic Boxes: Each item listen in a possible box first has a "chance" to be in the box, to increase the chance of an item being in the box, decrease the random x number. To decrease the chance of an item being in a box, increase the random x number.

        Secondly, if an item does spawn in the box, then we will set a number between 1 and random x.

        Example: FoodNutmix has a 25% chance of spawning and the box will contain between 1 and 3 FoorNutmix items.

         

        This way, even if the exact same mission plays twice, the boxes will be different.
