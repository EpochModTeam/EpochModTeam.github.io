        The installation from 1.0.5.1 to 1.0.6.1 didn't change. The files are the same and you could also watch a tutorial from 1.0.5.1.
        But there are already plenty of 1.0.6.1 install tutorials on youtube. I suggest you start there but normally a video is not needed as the installation is really simple.

        Because ArmA 2 has no seperate files for server only, you need to drop the whole ArmA 2 folder into your server root directory and after that the same with the ArmA 2 OA folder, overwrite files if asked.
        After that you just download the server files from here:Â http://epochmod.com/a2dayzepoch.php
        Just hit the green button on the right hand site for the server files.

        Now you can drop all files into your server (ofcourse to the right location). If you are not experienced with that, follow the official instruction on github:Â https://github.com/EpochModTeam/DayZ-Epoch/tree/master/Server Files

        Good luck! :)

        By the way, the overpoch installation is also still the same. Make sure that epoch is fully installed first and then just drop the @DayzOverwatch folder in your server directory aswell, add the missing addons to your mission.sqm, copy the overwatch key into the keys folder and edit your startup parameters (make sure that overwatch is loaded BEFORE epoch).
