        Hey,

        On line 5 you see:

         

        5 "PVAHR_"

        this means every thing that has PVAHR_ in it will get you instant kicked.

        So you wanna add:

         

        PVAHR_0_vvppiyql

        to not kicked list, so you simply change the line that is like:

         

        5 "PVAHR_" !"PVAHR_0_"

        to:

         

        5 "PVAHR_" !"PVAHR_0_" !"PVAHR_0_vvppiyql"

         

        so that means every thing is like !"" is expected.

         

        feel free to correct me if im wrong :)
