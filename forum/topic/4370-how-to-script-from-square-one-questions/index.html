        for starters you just download the PBO's to your computer, doesn't matter where.

        to extract the files you can't use winzip, you need a PBO manager or a tool which can decode and encode PBO files..

        Try this one for example: http://www.armaholic.com/page.php?id=16369

         

        If you want to learn to write scripts yourselfm check out this guide: Fockers Arma3 Scripting Guide 

        I know it says ArmA3 but ArmA2/DayZ is almost identical, just a few less functions but the scripts are the same!

         

        If you know the basics you can also find some nice tutorials here: http://killzonekid.com/category/tutorials/
