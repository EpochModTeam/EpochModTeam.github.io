        Right now you can't.  Have to wait for the 0.3 Epoch update to get support for Bornholm.  Bornholm uses an odd classname for roads, so the Epoch server doesn't recognize roads as being roads, so it spawns them in whatever open place it thinks is "safe".  Right now the only solution is for admins to move them manually using admin tools every day or two after they pile up.
