        Epoch does not use Bliss, and stores vehicles and buildables in object_data.  You can add buildings via the database, but you might find it more powerful to add them via the Arma2 3d Editor (ALT-E at the main menu) and then load the sqf via your mission file.  There are plenty of tutorials around on that.
