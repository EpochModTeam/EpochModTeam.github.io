        Epoch has an admin tool/AH that comes with it. You can access it by pressing F1 if you typed in your SteamID in the config that you can find here:

        \Arma 3\@epochhive\epochah.hpp

        adminMenu_Owner[] = {
	{"STEAMID","ADMIN1"},
	{"STEAMID","ADMIN2"},
	{"STEAMID","ADMIN3"}
};
