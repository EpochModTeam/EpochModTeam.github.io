        Mods/scripts will normally give you a guide in the thread/install files.

        Vehicles can be added into the allowedVehicleList in epochconfig.hpp:

        allowedVehiclesList[] = {
{"C_Offroad_01_EPOCH",15},
{"C_Quadbike_01_EPOCH",15},
{"B_Heli_Light_01_armed_F",20}, //pawnee
{"B_MRAP_01_EPOCH",20},  //standard hunter
{"B_MRAP_01_hmg_F",15},  //hunter HMG
{"O_MRAP_02_F",20}, //standard ifrit
{"B_Truck_01_Repair_F",20},  //hemtt repair
{"B_Truck_01_ammo_F",20},  //hemtt ammo
{"B_Truck_01_fuel_F",20},  //hemtt fuel
{"B_Truck_01_medical_F",15},  //hemtt medical
{"B_Heli_Transport_01_F",15},  //Ghost hawk
{"O_Heli_Light_02_F",15},  //armed orca
{"O_Heli_Attack_02_black_F",5},  //kajman
{"O_Heli_Transport_04_EPOCH",15},  // Taru lift
{"B_Heli_Transport_03_unarmed_EPOCH",15},  //Huron (chinnock)

        Adding mods to the server is almost the same as adding mods to your Arma game. Just upload the @whateveryourmodis folder to the server root directory, add the keys to the root keys folder and add the mod name to your startup parameters. You can add the mod names to your mission.sqm in the mission PBO to prevent people that dont have the mod from connecting. If people dont have the mod and CAN connect then they wont be able to see the mod. They will someone floating around if they are in a vehicle.

         

        To make whatever you added usable in-game you can set them to spawn randomly on the map by adding them to the allowed vehicles list as mentioned above (for vehicles) and for weapons if you want them to spawn in you can edit your loots.h file located in the A3_epoch_server_settings.PBO or by adding them to a lootspawner script.

         

        You can also add the class names to your trader list so you can buy and sell them.

         

        If you add armed vehicles that have rockets/missiles etc.. you will need to add Battleye exceptions or you will get kicked for using them.

         

        Heres a couple links to get you started. Good luck!
