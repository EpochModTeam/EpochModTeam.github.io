        You'll just set everything to the same code which isn't very secure or sensible.  What if you fall out with someone who has that code? Changing all those codes would get frustrating, quickly.  Personally, I think that safes and locked vehicles should only be accessible by the owner, once unlocked anyone can use it though, but I think assigning the users UID to the item is better than carrying around easily lost keys and codes you have to scribble down on paper (which you'll lose or forget at some point).  

         

        good luck, this problem has no cure ;)
