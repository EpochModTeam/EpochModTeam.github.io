        Finally i figuered out the problem.. not that google.com had any straith answers for me..

         

        what i needed to do was install trough steam

        arma 2 (cleane)

        arma 2 ao (clean)(after install go in to properties and beta and choose -beta)

        arma 2 beta (obsolete)

         

        run arma 2

        run arma 2 ao

         

        copy arma 2 to the server dir

        then copy arma 2 ao to the server dir

        download epoch_server files and copy them to the server dir

         

        copy the config example of the map u are going to use to the root of the server dir (remember the bat file aswell)

         

        change the hive and config file to match you db and all that

         

        and here is what i think did the trick. change the bat file before u lunch it, so u delete the /expantion/beta before the arma2aoserver.exe in the startup line.
