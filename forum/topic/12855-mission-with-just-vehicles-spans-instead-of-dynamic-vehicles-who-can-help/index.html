        You could always set up an SQL query to clean up unused spawned vehicles after X number of days. This is what I use to delete them after 3 days:

        DROP EVENT IF EXISTS removeUnownedVehicles;
CREATE EVENT removeUnownedVehicles
Â  Â  ON SCHEDULE EVERY 1 DAY
Â  Â  COMMENT 'Removes old unowned vehicles'
Â  Â  DO
DELETE FROM object_dataÂ 
WHERE
`CharacterID` = 0
AND `LastUpdated` < DATE_SUB(CURRENT_TIMESTAMP, INTERVAL 3 DAY) AND Classname NOT LIKE '%GunRack%'

        The reason I have the "NOT LIKE '%GunRack%' part is because I've found a number of gunracks on my server with a character ID of 0 and I'm not sure why, so I put that in there to be sure those aren't deleted by this query.
