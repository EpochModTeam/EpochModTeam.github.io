        Issue: Sometimes, land vehicles seem to be "stuck" when attempting to drive them. The vehicles appear to be in perfect condition and fully fueled, and you can even see the RPM dial spin up on the dashboard in first person view, but pressing any directional key fails to move the vehicle or produce any engine sounds.

        Â 

        How to replicate: This issue seems to happen occasionally. A vehicle that appears stuck may become unstuck after vehicle repair parts/kits are used on the affected vehicle or after a server restart. A vehicle that is made "unstuck" after using repair parts/kits or a server restart doesn't seem to become stuck again during the same server session. After the next server restart, it may become stuck again.

        Â 

        I haven't found if this bug affects helicopters.

        Â 

        Possible cause of this issue could be the vehicle being spawned slightly underground on server start when vehicles are loaded. I remember a piece of DayZ mod code which set a tiny setVelocity on each spawned vehicle to give it a small upward push to prevent a similar issue.

        Â 

        Edit: Issue encountered with A3 1.36/Epoch 0.2.5.2 on a server hosted on my own PC.
