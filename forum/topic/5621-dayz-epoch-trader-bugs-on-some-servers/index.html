        There seems to be a bug, a really strange bug, where traders that use skins that hold weapons, more often then not, stop working for some players.  
         

        There are other bugs with using trader skins that are exactly the same as the root skin that player skins are based on.
         

        One tedious fix is to log out somewhere near the trader, then come right back in. Most of the time, that's enough to allow you to now interface with the vendor. (This bug, incidentally, is linked mysteriously to the same reason why 2 players sometimes can't get into the same vehicle until they change clothes, or one of them logs out and then back in.)

        The "better" fix is have the server admins "reskin" all of the vendors that players are having trouble with to models that are both not using a weapon, and also not a root skin used by any other player clothing on the server.   (female skins seem to work the easiest for this.)

        hope that helps some.
