Completly forgot about this post.

Right now there is only one mission working and not completed yet.

But for your questions:

1: Mission will start at a random location from any ["NameVillage","NameCity","NameCityCapital"]. So it only spawn in random towns.

2: Depnding on mission, number of AI to defend thing can be customized, timer and skills can be changed for now.

This is the config file if you wonder what can be changed for now.

Spoiler 

//============================================////============================================//
//=====================================//-CONFIG START-//=====================================//
//============================================////============================================//


//==================================//
	//MAIN SYSTEM CONFIG
		MISSIONS_SYSTEM 	= 1;			// ENABLE OR DISABLE MISSION SYSTEM (Default 1=Enabled)
		ONLINE_PLAYERS 		= 1;			// NEED THIS AMOUNT OF PLAYER TO START A MISSION (Default 1)
		DEBUG_RPT 			= 1;			// ENABLE OR DISABLE RPT CONSOLE LOG INFO ABOUT MISSION EVENTS (Default 1=Enabled)
		SERVER_LOAD_WAIT	= 60;   		// WAIT TIME BEFORE STARTING MISSION SYSTEM AFTER A SERVER RESTART. (Default 300=5MINUTES)
		MISSION_CLEAN_TIME	= 120;			// HOW MUCH TIME TO WAIT BEFORE CLEANING UP MISSION OBJECTS AND THEN START ANOTHER ONE. (Default 900=15MINUTES)
		
//==================================// 
	//OPTIONAL CONFIG
		LOOT_TRACKER 		= 1;			// ENABLE OR DISABLE LOOT_CRATE MOVING MARKER ON THE MAP. (Default 1=Enabled)
		TOWER_BAR_TEST		= 1;			// ENABLE PROGRESS TIME BAR WHEN CAPTURING TOWER. VISUAL BAR BOTTOM RIGHT TRACKING TIME LEFT FOR CAPTURE.
//==================================// 
	//SPECIFIC MISSION CONFIG
		//-----------------------------//
		//-CAPTURE & DEFEND-//
		//-----------------------------//
			RT_CLAIM_RADIUS 		= 50;			// THE RANGE IN METERS AT WICH THE RADIOTOWER CAN BE CAPTURED BY AI OR PLAYERS. (Default 50=50METERS)
			TOWER_CAPTURE_TIME	    = 720;			// THE TIME NEEDED FOR PLAYERS OR AI TO CAPTURE A MISSION OBJECTIVE FOR BONUS +MP. (Default 900=15MINUTES)
			CLAIMING_GROUP			= 1;			// DROP AI TROOPS BY HELI DEFENDING RADIOTOWER IF PLAYER IS CAPTURING IT. (Default 1=Enabled)
			//ADVANCED CONFIG FOR CLAIMING_GROUP
				CLAIMING_GROUP_NUMBER   = 3; //(1,2,3)		// NUMBER OF GROUP TO SEND TO CAPTURE BACK THE TOWER. (Default 3) MAX 3.
				/*1*/AI_SF_CG_WAVE_1	= [180, 5, 0.10];	// [TIMER, AI_NUMBER, AI_SKILLS] SET AI_SF_CG_WAVE_1 SETTINGS. Default=[300, 5, 0.10]; 	5MINS+5(AI)
				/*2*/AI_SF_CG_WAVE_2	= [600, 5, 0.10];	// [TIMER, AI_NUMBER, AI_SKILLS] SET AI_SF_CG_WAVE_2 SETTINGS. Default=[600, 5, 0.10];	10MINS+5(AI)
				/*3*/AI_SF_CG_WAVE_3	= [900, 5, 0.10];	// [TIMER, AI_NUMBER, AI_SKILLS] SET AI_SF_CG_WAVE_3 SETTINGS. Default=[900, 5, 0.10];	15MINS+5(AI)
			//

//============================================////============================================//
//======================================//-CONFIG END-//======================================//
//============================================////============================================//

Nice,

I have some ideas

1) Hostage Mission :  Close quarters only battle.  Ai placed inside of a few buildings that players have to search in order to find and rescue the hostage(s) Maybe with a time limit until the AI kill the hostages? Place civs in with the AI to prevent grenades rockets being used - kill a civ = no loot reward/mission failed.

2) Convoy Mission . AI in armed vehicles and on foot. Could be either a rolling convoy or static (broken down vehicle?) Once player engages mission the AI call for reinforcements - Air Land or paratroops.

3) Spy Hunter . AI civilians placed in a town/city and one of them is a spy, the players have to find which one is the Spy and execute him/her. If player kills civilian = mission failed. Players could ask civilians questions?

4) Clear the oil rig - or some kind of sea based, or boat based mission.

5) Racing Mission . Ai and player race to a location, first one there wins.

6 ) Player Base Attack . Random player base is attacked by AI waves for a set time. Waves of AI with different types of loadout try to destroy the player base and kill the players. Snipers, Heavy weps, rocket units, maybe a vehicle. Maybe chose base by players that are online at the time.

7) Manhunt .  AI choose random player and hunt him/her for a set time. If player survives the timer = mission complete, if player dies in the timer mission fails.

8) Designated targets . Players have a time limit to destroy 2 or 3 buildings guarded by AI and vehicle patrols.

Hope these give you some inspiration! Keep up the good work.
