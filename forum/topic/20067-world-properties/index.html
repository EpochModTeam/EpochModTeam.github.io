        Keep in mind that the mod is in Alpha.  At this point we are only supposed to be testing it, not playing it.  They have to keep all the variables the same across all servers so they can have a consistent picture when troubleshooting issues.  What's more, if they were to crank up the build limit at this point and then continue development it would be tough for them to know for sure if it was the higher build limit that's affecting performance or some of the new code they just added.  It's highly likely that the build limit will be raised once they get closer to final release, but until then we just have to roll with it.  If the build limit problem becomes a consistent issue they could always just wipe the database and let people start over.
