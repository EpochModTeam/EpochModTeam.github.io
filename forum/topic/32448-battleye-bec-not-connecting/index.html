        So I have my arma 3 epoch server up and running working fine and people can play, I try to load BEC and I keep getting No Valid Response Try to Reconnect. I tried with a RCON tool and it's still not connecting to the server.

         

        Firewall Is completely off and arma 3 server running locally on the same pc.

        so it launch and does not complain about the path just saying cannot connect to server. I seen alot of post regarding this on forums and this one too and most of the stuff that people have for fixes still does not work with me. Any help would be appreciated

        If BEC is running on the same machine the game server is on your IP should be 127.0.0.1, which is the standard IP for localhost.  If it's on a different computer across the internet, then you may have the wrong IP address listed, since the 192.168 subnet is considered an "internal" IP and does not get routed online.

        yeah I figured it out, so since I am using TADST to launch my server under the TADST it has a folder battleye there.

         

        Here is how to:

        1. Drop your BEServer.dll, BEServer.cfg and bans.txt (and other txt file setup in your Config.cfg of BEC) into your Arma3\TADST\default\battleye folder.
        2. Close your arma 3 server and start it again. 
        3. Wait a bit or whatever timeout you put into your BEC Config file
        4. Then start BEC it should connect right away

        I don't know why TADST have his own Battleye folder, Arma 3 has one too and the SC folder has one too..... Should I delete those ?
