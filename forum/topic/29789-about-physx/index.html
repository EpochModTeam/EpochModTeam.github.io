        Well, here is my belief about whats going on.. Take this from the standpoint from an Admin of servers.

         

         

        With the old system you could build eloborate bases which did look cool. Trust me, I build one heck of an admin base that looked like a friggen castle.

         

        The issue with doing this, was that it was not bound by phsics and was ever so easy to build Ginormous bases. This in turn cluttered up servers quickly, allowed for unbielvable constructions that could not exsist in the real world. Ever go on a server and see a base just floating in mid air? Just retarded IMHO.

         

        With the new Arma3 Epoch building system, it was IMHO designed to prevent the hovercraft bases, add complexety which adds time to build requirements and thus reduces the amount of worthless bases on servers.. 

         

        As I said being an admin Going through and deleteing bases that were abonded because its just easy to start fresh then to remove, is why the new Physic system is a plus.

         

        I will say one thing, the major Con with the building system is that people are lazy. I've been playing on the DonkeyPunch server, people there have no will to build a base but rather just board up exsisting buildings. I.e. the construction building. Thus it make it impossible to get a good looting.

         

        What I suggest to the Devs, if there is a frequency jammer near a High loot area and player building present that loot no longer spawns. This prevents people from basicly getting Free stuff with little to no risk.

        You can still have a floating base, it just takes more time to do it. So this new system doesn't really solve that problem.
