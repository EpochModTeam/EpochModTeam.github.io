        The boxes are not technically preloaded with loot. The loot spawns in them once you open them. Just like the loot containers in buildings. Like chairs, coolers and couches.

        Their contents and chances of having particular loot are all controlled in the loot tables.

        It is simple to spawn them. Here is the code from Epoch that spawns the ship wrecks. 

        private ["_item","_marker"];
{
	// 20 percent chance for loot to spawn
	if ((random 1) <= 0.4) then {
		_item = createVehicle["container_epoch", _x, [], 0, "NONE"];
		_item setMass 220;

		if (EPOCH_SHOW_BOATLOOT) then {
			_marker = createMarker[str(_x), _x];
			_marker setMarkerShape "ICON";
			_marker setMarkerType "mil_dot";
			// _marker setMarkerText "Shipwreak";
			_marker setMarkerColor "ColorOrange";
		};
	};
} foreach (getArray (configFile >> "CfgEpoch" >> worldname >> "containerPos"));

        To change what loot is inside the container, you have to dig into the loot tables. And that will change the loot content for all shipwrecks as well.

        Okay, here we go.

        @EpochHive\addons\a3_epoch_server.PBO is the PBO you will be looking in.
