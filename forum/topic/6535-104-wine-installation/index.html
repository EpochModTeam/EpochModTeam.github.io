        I ran into the same issue but the fix is easy:

        Â 

        If you start the server without the redist package wine will tell you which dlls are missing:

        • MSVCP120.dll
        • MSVCR120.dll

        Just copy the missing 32-bit dlls(either from your windows or download them from the web) into your arma-root and it will run.

        Â 

        GL&HF!
