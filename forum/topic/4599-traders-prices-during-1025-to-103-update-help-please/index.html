        Posted December 10, 2013

        Ok... I'll bring that back to the top because I am kind of a server noob.

         

        I have downloaded the manual server side zip file, and have found the 1.0.3 sql file, am extremely new to any editing and am using HeidiSQL...

         

        Please help... Like I said above, the majority of the editing/script additions to the server I am fine to add back in, but would absolutely be devastated to lose hours upon hours of custom trader pricing and such.

         

        Open HeidiSQL and connect to your database. Click on "Query" - doesn't matter what table you do this in. Go to the server files and find the file called 1_0_3_Updates.sql and open it with a text editor. Select all and copy. Go to the Query tab and paste the contents from the .sql file. Hit F9. 

         

        Also, see this topic: http://dayzepoch.com/forum/index.php?/topic/4528-103-database-update/ as it includes the .sql code for the new vehicles as well, as the dev's left them out of the updates.sql file.
