        hey guys, i had this problem where my server wont appear in Dayz commander and when i added the ip to the favorite it just shows the ip and 100000 ping and nothing else. so to fix this i asked help from the server provider and all you have to do is open up cmd and enter del %appdata%\..\.\Local\DayZCommander\settings.xml  

        or go to appdata and local and dayzcommander and delete settings.xml

        this fixed dayz commander for me and all the servers show up without a problem now

        Dayz commader is just silly with saving...

         

        After the lastest patch it searches for the games on steamports, which are mostly defaultports (27016)

        When adding a favorite from the list it still saves the gameport(2302)

         

        Simple solution:

        Save your favorits by steamport:

         

        Ser.ver.i.p:27016

         

        this should fix it for the most of you

         

        If you are not using the default ports:

        load the list, make sure u deleted all your favorits using gameports, browse your server

         

        then place your mouse over your server -> shows up ip and steamport

        add this ip and steamport manually
