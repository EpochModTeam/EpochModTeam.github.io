        I have been investigating performance issues on my server and have narrowed down lag spikes to the LSDeleter.SQF in the LootSpawner script I use (it happens every 60 seconds). Disabling Lootspawner stops the spikes altogether.

        What I would like to be able to do is stop using lootspawner but I have it customised in a way that currently isn't possible with the default loot spawn system (even with the additions already in 0.3).

        Would it be possible to add in the functionality of having server customisable building class lists set to spawn specific classes of items?
