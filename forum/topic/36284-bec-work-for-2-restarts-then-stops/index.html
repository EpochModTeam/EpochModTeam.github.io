        If you are restarting them at the same time, try restarting one an hour later than the other.

        I run a Chernarus and Altis server on the same box and don't have any problems running them.

        If problems still persist, double check the schedule.xml of the altis server and make 100% certain there is not a typo or a bad command somewhere.
