        in your mission init.sqf, find -

        MaxVehicleLimit = NNN;

        and lower it :)

        if, for some reason, it ain't in your init, just paste that line in there, just under -

        // DayZ Epochconfig
spawnShoremode = 1; // Default = 1 (on shore)
spawnArea= 1500; // Default = 1500

        then delete some vehicles off your server :)

        no, it doesn't delete any. it just lowers the maximum amount of vehicles allowed on the server. once the server hits this number, it just stops spawning more vehicles in. it'll never actually affect any vehicles.

        Â 

        i'd also suggest running this in your database -

        CREATE EVENT DeleteOldVehicles
ON SCHEDULE EVERY 1 DAY
COMMENT 'Removes unused vehicles'
DO
DELETE FROM `object_data` WHERE `LastUpdated` < DATE_SUB(CURRENT_TIMESTAMP, INTERVAL 14 DAY) AND `CharacterID` > 0 AND `Classname` NOT IN ('VaultStorage','LockboxStorage','VaultStorageLocked','LockboxStorageLocked') AND `Inventory` <> '[]' AND `Inventory` IS NOT NULL

        change the 14 to any number of days you like. it'll delete vehicles that haven't been touched in N days.
