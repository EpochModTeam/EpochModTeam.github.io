        You don't have to use the launcher :) download it via Bittorrent Here

        Once downloaded place the @Epoch folder inside your ArmA 3 directory (C:\Program Files (x86)\Steam\SteamApps\common\Arma 3)

        Then go to Steam > Library > right click ArmA 3 > select 'properties' > click 'set launch options' > add the following :

        -mod=@Epoch

        Launch the game and filter for Epoch servers :D Have fun
