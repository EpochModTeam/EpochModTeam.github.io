        Start with verifying the integrity of your files in Steam for ArmA 3. Then you can check which version you are running when you launch ArmA 3, on the main menu should be a version somewhere in the corners. Once you get A3 on the right version, then you can use BittorrentSync to make sure you have the current @Epoch files and then move them to the Program Files > Steam > Steam Apps > Common > ArmA 3 folder (or if you already have @Epoch, into the @Epoch folder.)

         

        I hope this helps!
