        Hey guys,

         

        I've been playing unleashed the last days, just for fun...

         

        Then we changed back to epoch, but I actually missed the idea of doing something specific.

         

        Unleashed actually had classes like medic, hunter, engineer and stuff. Also you could level them up by doing things the class was made for.

         

        So I had the idea to combine this feature with the trading system of epoch.

        The classes would be kinda like the ones unleashed used so:

         

        A medic: does medic stuff tto rank up

        A hunter: kills people/zeds/bots to rank up

        A engineer: repairs or builds things to rank up

         

        basically I wouldnt like to seperate these classes, so each player would get something like the humanity counter just for the exp collected in each of these jobs

         

        at certain values you are able to get some abilities:

         

        Medic: able to give selfbloodbag // less change to get ill // dont gets cold as fast as normal // craft medical stuff

         

        Hunter: able to convert some kinds of ammonation 200 round saw to stanag // hold breath some longer // doesnt get detected by zeds while shooting within range wich is smaller then normal (normal 100m atrackting range -> improved to 85m

         

        engineer: build things up in less time // able to carry more weight // maintain costs less // reapairing fails less often (salvage) // able to take fuel from other via jerrycan in one process

         

        just smaller things like those

         

        for traders I would add some special traders able to enter at a certain value of each of those skills

         

        medic trader selling medical stuff for cheap

        hunter trader selling low value weapons and ammo for cheap and some sandbag stuff or ghille suits

        engineer trader selling vehicle parts for cheap and toolbelt items 

         

         

        I guess this would implement some long term achievements to the game besides from build up bases and isnt really overpovered

        a global variable to give the admines the chance to decide whetere they want it ot not would be nice, since some people wouldnt agree with that idea anyways

        Thanks for reading
