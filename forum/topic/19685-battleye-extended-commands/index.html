        This is what I use to lock the server 5 minutes before restart. I have 4 hour restarts:

        <job id="1">
<time>035500</time>
<delay>000000</delay>
<day>1,2,3,4,5,6,7</day>
<loop>0</loop> 
<cmd>say -1 Locking server to prevent new players from joining.</cmd>
<cmdtype>0</cmdtype> 
</job>
<job id="2">
<time>035500</time>
<delay>000000</delay>
<day>1,2,3,4,5,6,7</day>
<loop>0</loop> 
<cmd>#lock</cmd>
<cmdtype>0</cmdtype> 
</job>
