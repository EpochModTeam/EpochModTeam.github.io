        If your server is running on port 2302, then your SteamQueryPort is 2303 and your SteamPort is 2404.  It's always query port = +1 and steam port = +2.  Once you make the changes and restart let your server run for a while.  It can take up to an hour for the Steam server browser to update.  If you have trouble finding your server in DayZ Commander, try finding it based off of the steamqueryport.  Or you could also try DayZLauncher instead.  It tends to be more responsive anyways.

        If your router requires you to do port forwarding then yes, you will have to open all relevant ports, including your server port, steamport, and steamqueryport.  If you're running the server on your home machine and you intend to play on the same machine, you can certainly do so.  But getting others to connect to it from the outside world isn't always easy.  Some routers can be rather picky about what traffic it allows inside the internal network.
