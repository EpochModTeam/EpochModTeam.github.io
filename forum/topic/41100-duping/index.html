        It is because DLC machineguns and MX SW spawn with bipods since Marksmen patch. Then again I do not see as a super big issue though if one has 10 navids etc, everytime removing excessive bipods from a storage is pretty annoying when you want to put some items inside :) Temporary fix would be removing attached bipods from those guns manually in loot table so that bipods only could be looted in regular Epoch loot, not from guns itself, however, a bipod on machinegun is pretty necessary feature so it makes sense that Bugemia did it this way by making them default on a gun and making them removable (unlike with Lynx or LRR for example). And of course, Epoch considers gun and a bipod as two separate entities so you get duping after each restart (just guessing here).
