        Are you sure you have turned events on? 

        Click the main database tab and execute these commands to see:

        SET GLOBAL event_scheduler = ON;
SET @@global.event_scheduler = ON;
SET GLOBAL event_scheduler = 1;
SET @@global.event_scheduler = 1;
