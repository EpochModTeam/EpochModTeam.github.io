        You're probably are using Blurgaming AH or something similar.

         

        Blur scans the player's folder.

         

        While scanning, it scans for 'Rustler'

         

        Rustler is a listed in the BattlEye scripts.txt along with some other hack names.

         

        How to fix:

        1 - Take out Rustler (around line 25 it starts) and all other hack names from scripts.txt

        2 - Reload scripts.

         

        Done :)
