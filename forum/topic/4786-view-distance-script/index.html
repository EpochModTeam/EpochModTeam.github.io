        Ok i see another mistake..

        viewDistance (http://community.bistudio.com/wiki/viewDistance) returns a number and you are testing if it is false. Which makes no sense. Unless you have instantiated a variable viewDistance to be a boolean which I don't see why you would do as that may cause conflicts.

        Your if statement currently says if (viewDistance == false) which will always fail as a number is not equal to false and hence your if statement block will not run.
