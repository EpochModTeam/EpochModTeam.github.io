        @Franky

        go to WAI/config.sqf

wai_radio_announce			= true;								// Setting this to true will announce the missions to those that hold a radio only

        I think he wishes to change the way the message is shown on the screen.

        @Franky

        in your WAI files navigate to

        compiles\mission_winorfail.sqf

        In this file you will be able to edit the messages

        E.G.

        Making a chat msg when a mission starts

            [nil, nil, rspawn, [_msgstart], {
                if !(isServer) then {
                    _msg = _this select 0;
                    systemChat ("(AI MISSION): "+str _msg+"");
                };
            }] call RE;

        Place that below 

        "} count allUnits;"

         

        Hope this helps
