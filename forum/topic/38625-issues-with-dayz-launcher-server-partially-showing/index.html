        We had this issue in the beginning. What I did:

        Made sure my port for the server was open

        Added this in my config.cfg and made sure those ports were open aswell.

        steamport = 2300;
steamqueryport = 2301;

        Also we had our Arma and Epoch versions showing in the server name but we don't have them anymore. I don't think they are needed as our server shows in the launcher.

        For the first 30 minutes our server didn't show up either so we used Armas direct connect to connect and suddently it showed up and has been there ever since. Try to keep your server up normally and connect through Arma server browser.

        Do you have any errors in your .rpt?
