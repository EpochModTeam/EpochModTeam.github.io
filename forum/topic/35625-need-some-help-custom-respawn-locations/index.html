        Here you go

         

        1)Open your server_playerSetup.sqf (located in dayz_server\compile\server_playerSetup.sqf)

         

        Find this line of code

        dayzPlayerLogin2 = [_worldspace,_state];

        Change to

        dayzPlayerLogin2 = [_worldspace,_state,_randomSpot];

        2)Open your init.sqf (located in Mpmission\init.sqf)

         

        Add this block of code at the very bottom

        ///*** Based Custom Spawn Locations ***\\\
p2_newspawn = compile preprocessFileLineNumbers "Scripts\newspawn.sqf";
waitUntil {!isNil ("PVDZE_plr_LoginRecord")};
if (dayzPlayerLogin2 select 2) then
{
player spawn p2_newspawn;
};

        3)Make a folder named Scripts in the root of your MPmissions. Make a sqf file named newspawn.sqf and place that inside your Scripts folder

         

        Place this code inside the newspawn.sqf

        if (!isDedicated) then {
		sleep 1;
		
		if ((getPlayerUID player) in ["11111111111111111","22222222222222222"]) exitWith {
		//## Custom Spawn Location 1 - Format: [X,Y,Z] (Z = Height AtTerrainLevel)
			player setPosATL [473.234,344.694,0.00];
		};
		
		if ((getPlayerUID player) in ["0","0"]) exitWith {
		//## Custom Spawn Location 2 - Format: [X,Y,Z] (Z = Height AtTerrainLevel)
			player setPosATL [465.358,345.407,0.00];
		};
};

        Place the Players UID in place of the "111111111","222222222".

         

        Change the world space in  player setPosATL [465.358,345.407,0.00];  to the spot you wish them to spawn at.

         

        Note: Z = Height AtTerrainLevel. If you have halo spawn on you need to change this to   player setPosATL [465.358,345.407,2000.00];  If you don't they will instantly die when they spawn.

         

        You can add more spawn locations by simply copy/pasting it again like this

        if (!isDedicated) then {
		sleep 1;
		
		if ((getPlayerUID player) in ["11111111111111111","222222222222222222"]) exitWith {
		//## Custom Spawn Location 1 - Format: [X,Y,Z] (Z = Height AtTerrainLevel)
			player setPosATL [473.234,344.694,0.00];
		};
		
		if ((getPlayerUID player) in ["0","0"]) exitWith {
		//## Custom Spawn Location 2 - Format: [X,Y,Z] (Z = Height AtTerrainLevel)
			player setPosATL [465.358,345.407,0.00];
		};

		if ((getPlayerUID player) in ["0","0"]) exitWith {
		//## Custom Spawn Location 2 - Format: [X,Y,Z] (Z = Height AtTerrainLevel)
			player setPosATL [465.358,345.407,0.00];
		};

		if ((getPlayerUID player) in ["0","0"]) exitWith {
		//## Custom Spawn Location 2 - Format: [X,Y,Z] (Z = Height AtTerrainLevel)
			player setPosATL [465.358,345.407,0.00];
		};
};

         

        Please keep in mind i have not used this script in a long time (8 months or more) and it might not work perfect every time. This all matters what other scripts you have running. However, it should work just fine. One thing i have seen in the past was once in a while a player part of this code logs in and gets spawned again at the location with no gear loss.

         

        If this does not work as intended/wanted i recommend giving ESS a real shot as it can do this and more. If you don't want all the features of ESS you can disable them all and only have spawn selection. I would even help you get it working just how you want.

         

        Hope this helps.
