        Impression a girl with a Epoch server? Haha!

         

        1. Rent this package: https://www.vilayer.com/gameinfo/dayz-package.html

            - Wait till you get a "DayZ Package Provisioning Completed" mail from Vilayer.‏ ( Follow login instructions from that mail )

         

        2. "First Steps after renting": http://www.youtube.com/watch?v=G7SmEGjTaio

            - In this video he is installing the basic Dayz Chernarus. You want Epoch, so install Epoch, you will see it for yourself. 

         

        3. DataBase, MySQL install: http://www.youtube.com/watch?v=odSm2XJC814

            - This is to save your Base etc. etc.

         

        This all will take max. 2 hours for a beginner. This is without custom scripting, that is a "next" level. But you will learn that soon, after 1 script succesfully installed you will be able to install all. Then you will say, was that all?

         

        Just take your time, and i work too.. but not 24/7.
