        Server update notes for 0.3.0.2

        Asked by vbawol,
        April 8, 2015

         Share
        Followers 0

        vbawol

        Posted April 8, 2015

        vbawol

        Posted April 8, 2015
        1. Update client files to 0.3.0.2.
        2. Update both server pbo's (a3_epoch_server.pbo, a3_epoch_server_settings.pbo) and epochserver.dll
        3. Remove epoch0300.bikey and replace with epoch0302.bikey.
        4. Add the line LogAbuse = 1 under InstanceID in the EpochServer.ini.
        5. Make sure pcre3.dll is unblocked and in your Arma 3 root folder.
        6. Update BE filters.
        7. Update redis-server.exe from the DB folder to the latest stable build 2.8.19.
        8. Update MPmissions files (description.ext updates).
        • Hippeh, Darth_Rogue and Salutesh
