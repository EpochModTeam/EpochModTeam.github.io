        Had this same issue Bud! do not use overwatch_server. You only need the .bikey

         

         

        I see a couple of things. 

        Script z\addons\dayz_server\missions\DayZ_Epoch_11.Chernarus\mission.sqf not found &

        Script z\addons\dayz_server\init\server_functions.sqf not found

         

        This tells me you either you did not pack your .pbo right or the direct is all screwed up.

        To be honest, start from the begging all over again. 1 step at a time. 

        Once you have a fresh new Epoch server running with no errors.
        Only copy and paste the overwatch.bikey into the key folder.
         

        at this time do not play with the epoch server .pbo!

        in your server launch perimeters add to -mod=@DayzOverwatch;@DayZ_Epoch;@DayZ_Epoch_Server;"

        (make sure its this way or problems will happen)

        That's all you need to really do.

        To add Overwatch items to trader, that's a little more detail and can be frustrating.

        So now that the server is up!

         

        ​Load your Arma 2 OA and go into expansions.  Of course make sure @DayZ is shut off.
        you should it like this:
        @DayZEpoch
        @Overwatch

         

        Restart and hopefully you get into game.
        If you see your server and your client crashes... Then you need to make sure you have the right steam patch or 1.8 through dayz commander.

        Once you get that resolved, YOUR only 1 step away from making it an Overpoch Origins server  :)         

        If you need help mesg me.          I am a infistar user for home network....
