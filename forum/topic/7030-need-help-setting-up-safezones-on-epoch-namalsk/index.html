        yaay. finally it works with 

        Even Better Safezones from Maca134

        http://epochmod.com/forum/index.php?/topic/4691-even-better-safezones/

         

        1. you have to go to Namalsk map http://dayzdb.com/map/namalsk#6.042.080

        2. download coordinate converter for namalsk http://berdyev.com/db/60

        3. on the map hold your mouse over a trader position e.g. 043 080 (first is x, second y)

        4. use the converter and you get coords e.g. Coordinates are: [0,[4300,4800,0]]

        5. just copy paste the coords into the safezone\init.sqf from maca´s mod

        _safezones = [
	[[11447.91,11364.536],100, "Klen"], // Klen

        to this

        _safezones = [
	[[4300,4800],200, "Test"] // Test

        ----------

        i edit the range to 200 and i got a big red marker on the map
