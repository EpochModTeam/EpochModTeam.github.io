        Hi folks, we recently converted to Epoch and so far teh feedback has been very positive and we will be supporting the mod as long as it's available.

         

        We run very close to the standard release version having added only some basic map updates and AI. Here is some feedback from our members comments;

         

         - Would be nice to see excelsior bridge mod added to the official Epoch mod. Its basically a well edited bridge linking Skalistky island to the main land. 

         

         - Some of the Epoch custom buildings (tents etc) dont spawn gear. This is probably because of the editor used to add them. 

         

         -Some of the traders had exploit sale values added (cant remember what they were now off the top of my head sorry) but it was something ridiculous like a gun that could be easily found or acquired and then sold for 10x 10oz gold. Ill see if i can research this better - sorry I did see it myself in the trader_date table but didnt make a note of it as i changed it :S

         

         - More custom weapons would be nice if they were added, particularly handguns!

         

         - Better controls for deleting vehicles. We have loads of vehicles that are next to useless that arent deleted by our auto  or manual clean ups because they arent 100% destroyed.

         

         - looking forward tot he web interface for the trader adjustment! 

         

         - I personally as a server admin prefer your method of making map updates. Using the missions file is much easier than the previous method i used on regular day z with injecting code to the tables themselves.

         

         - The ability to add server water marks/rolling messages! This would be a massive help for us server owners!

         

         - WHolesallers need a bit more work as they are a bit of an exploit at the moment and too easy to get gear from.

         

         - Base building is fantastic but would be nice to see it expanded further! 

         

         - player Knock out  scripts would be fantastic!

         

         - Ability to add short night cycles for those of us who dont have access to the dayz_code.pbo

         

         - Ability to not get gear from locked cars(i know this is a big ask!)

         

         

         

        So far it's my favourite mod and im very happy with it well done!
