        I want to install a private LAN server for myself so I can use Admin Tools and build bases to test and see how they would look finished so later I can build them on the server I play.

        But when I join my server I get this message after it's done loading:

         

        "This server is running an incorrect version of the server-side application. You can't play on this server. If you are the server administrator please contact Epoch staff."

         

        Here are the steps I did.

         

        1. Installed Arma 2 and Arma 2 OA (Steam)
        2. Copied AddOns folder from Arma 2 to OA folder
        3. Installed Epoch client AND server. (1.0.4.2a)
        4. Installed MySQL server and Navicat for MySQL. Created a database named "dayz_epoch" and imported the epoch.sql file.
        5. Copied "instance_11_Chernarus" from Config-Examples to root folder (arma 2 oa) and the same with "DayZ_Epoch_11.Chernarus".
        6. (Really I don't think this is right.. I just didn't know where these folders went).
        7. Edited HiveExt.ini on "instance_11_Chernarus" and added the database username and password
        8. Launched the dedicated server with "DayZ_Epoch_instance_11_Chernarus.bat".

        Any help would be gladly appreciated, thanks in advance.
