        Specs seems like they should net solid 60 fps.  Gonna say right off the hop its could be just bad luck with servers.

         

        Things I suggest.

         

        #1 In your hidden files c:\users\your name\appdata\local\arma2OA\  there's a file called ArmA2OA.RPT this file keeps track of local errors while running the game.  Until you get this fixed I recommend that before each time you launch the game delete this file so that it will make a new one for you to examine for errors.

         

        So each time you connect to a server check that file for errors.

         

        #2 Try running steam as administrator.  Try running Arma 2 just to make sure steam doesn't say that it is the first time it is being run.  Also try running Operation Arrowhead as admininstrator.  Running as admin might help with the FPS and if steam for whatever reason thinks arma 2 has never been run then there will be errors.

         

        #3 Keep trying different servers and see if you get different results.  Hopefully you will find that it was a server issue and nothing to do with your computer.

         

        #4 You could also try special launch parameters but I don't know exactly which ones to recommend.

         

        Not sure what else to say.
