        NOTES ON THE CODE BELOW

        1. You launch MrManager.bat at 'feasible client connection time'.
          For example, 3 minutes after server restart looks good to me.
          To put it to numbers, for a 0900 server restart, you double click on your MrManager.bat on your client computer and it basically takes your Arma and connects it to your fav server.
          Then automatically kills it at intervals -whenever you configure it to- and it keeps relaunching>killing>relaunching...
          Â 
        2. Preconfigured values below will probably need tweaking:
          1. Code below autoreconnects every 180 mins -- simply change 10800 secs timeout value according to your needs.
          2. If you are not using 1920x1080 resolution you will need to change the area that it 'clicks' on your screen.
            In order to do this, either install AutoHotkey or find an alternative software that can give you screen coordinates.
          3. Waits 1 second when it 'clicks' on Steam
          4. Waits 50 seconds when Arma2OA is launching & auto connecting to the server
          5. Waits 10 seconds for pskill to kill GoToGuy
          6. Code is for Overpoch, remove the Overwatch bit if you use standard Epoch.
          7. You will need to adjust paths on your system (namely: Steam.exe, pskill.exe, GoToGuy.ahk).
          8. You will need to download free pskill.exe from Microsoft website and place in the same directory with the two text files copied below. They rely on pskill to kill the existing arma2oa process.
            Â 
        3. The .AHK you see there is AutoHotkey - a freely available scripting/macro language for windows. Similar to AutoIT if you know that one. You do NOT need AutoHotkey installed.
          It's no problem for me to install a free software piece if it's doing what I want but some people want to keep their software packages list minimal.
          If you're in that boat, you can get this AHK modified for your system and then converted to an executable program file (.EXE) by a trusted friend.
          Obviously they will need to have AutoHotkey installed to do this.

          >> EDIT: Tested & working in fullscreen mode <<<

        GoToGuy.ahk (connector / muscle)

        ;------------------------------------------------------------------------------------------------------------------------------------------------------
;
;
;		WHAT DOES GoToGuy DO?.
;
;		GoToGuy knows where your Arma 2 OA wanna go (i.e.: your server).
;		GoToGuy knows how to get there.
;		GoToGuy has the connections (i.e.: TCP/UDP) and the resources (i.e.: Steam.exe) and finally the ability to manage these (i.e.: wait, click, wait)
;		GoToGuy is well established in business but he doesn't wanna talk to you - sorry!
;		GoToGuy however is more than happy to work for you as long as all communication (and payment) goes through secure channels (i.e.:  MrManager)
;
;
;		>>> tl;dr GoToGuy launches your Arma2OA and connects to your server <<<
;
;
;------------------------------------------------------------------------------------------------------------------------------------------------------
;		NOTE:	GoToGuy is actually not one guy but two! 
;		They are single-egg twins, luckily for you, they're both available for subcontracting via MrManager.
;		You can pick either GoToGuy.ahk or GoToGuy.exe.
;------------------------------------------------------------------------------------------------------------------------------------------------------
;
;
;

;
; Hello,  MrManager!
;

CoordMode pixel,screen
CoordMode mouse,screen
SendMode Input  ; Recommended for new scripts due to its superior speed and reliability.

; Play the sexy startup sound
SoundBeep, 750, 500  ; Play a high pitch beep for half a second - cuz GoToGuy likes 80s style! (he's in his 40s)

; Launch Steam, run our favorite game, connect to our favorite server
Run E:\Steam\Steam.exe "steam://rungameid/33930//-nosplash -world=empty -skipIntro -noPause -winxp -cpuCount=6 -exThreads=10 -connect=SERVER.IP.HERE -port=2302 -mod=@DayzOverwatch`;@DayZ_Epoch`;"
Sleep, 1000  			; Wait a sec for Steam's "Allow game launch?" dialogbox to appear

MouseMove 927, 534, 0	; On my 1920x1080 resolution screen, these are the coordinates for "OK" button, in Steam's "Allow game launch?" dialogbox.
SendInput {click}		; Yes, this application launch is approved.
Sleep, 50000  			; 50 seconds -- this is roughly about what it takes (on my PC) for Arma2OA to launch and connect to the specified server. 
						; Remember to allow a few extra seconds, as your PC and/or network may be under heavy load at times

MouseMove 1681, 960, 0	; We should be at the Lobby by now; let's click on OK to spawn in
SendInput {click}		;

;
; Goodbye MrManager and remember you can call me anytime but for best performance please no more frequently than $server_restart_cycle :)
;
Exit
;
;
;------------------------------------------------------------------------------------------------------------------------------------------------------

        @echo off
:: Turn off screen text messages


:: ------------------------------------------------------------------------------------------------------------------------------------------------------
:: 
:: 
:: 		WHAT DOES MrManager.bat DO?.
:: 
:: 		MrManager knows your restart cycle (e.g.: My server restarts ever 180 minutes).
:: 		MrManager knows when exactly your server restarts (e.g.: My server restarts at the following times of day: 0000, 0300, 0600, ...., 2100).
:: 		MrManager is good with Maths! He can add +3 mins to any time-of-day you give him & calculate the safe time to kill Arma2OA application.
:: 		MrManager can be ruthless! When he deems necessary, HE WILL KILL without blinking an eye!
:: 		MrManager is a master peacemaker also - he will not shy away from CALLING WHAT HE KILLED just 60 seconds ago! ;
:: 
:: 
:: 
:: 
:: 		>>> tl;dr MrManager runs the operation, makes your Arma2OA disconnect & reconnect <<<

:: 		--------------------------------------------------------------------------------------------------------------

:loop

	:: Here's an innovative idea... 
	:: Let's call GoToGuy and tell him:	"We were not happy with the last time but this must be your lucky day! Here, we're giving you another chance... Go!"
	start "GoToGuy" "D:\BusinessAssociates\GoToGuy.ahk"

	:: Wait for 180 mins

	timeout /t 10 >null

goto loop
