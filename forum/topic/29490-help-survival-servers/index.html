        Hi,

         

         

        It should be in the server browser once someone joins the server and start the mission going, Just need to go into the server and come back out and then look for your server in the browser. But you can not even remote connect so its could be releated to server ports if the server says "running" or its a actual server error and the provider should look at the server files.

         

         

        Check the RPT logs on the server. Since its a dedicated server here is my command line for my dedicated server:

         

          -noPause -noSound -mod=@Epoch;@EpochHive; -config=C:\TCAFiles\Users\User\Service ID\SC\config.cfg -ip=Removed IP -port=2302 -profiles=SC -cfg=C:\TCAFiles\Users\Username\Service ID\SC\basic.cfg -name=SC "-BEPath=C:\TCAFiles\Users\Username\Service ID\BattlEye"

         

         

        Regards,
