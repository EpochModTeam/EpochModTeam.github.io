        @Schalldampfer

        Ok this is the process:  you start your server with:

        1-SERVER LAUNCHER: this bat call the arma2oa server, wait 25 seconds to be sure that the server is loadup and call the BEC_CONTROL.bat: This file call to Bec.exe (if  arma2oa server is not running bec cant loggin in.) While BEC_CONTOL.bat is logged with your server (This is very important) you will see that the windows tittle change his name.

        After 3hs the scheduler.xml calls the SERVER_SHUTDOWN.bat. : This file find the arma2oa and shutdown. When arma2oa is down Bec automatically logged out. (His windows tittle changes again by "BEC_CONTROL" then SHUTDOWN recognize this name and kill the dead process.. call the SERVER_MAINTAIN and arma2oa server, and bec again.

        Here i upload this video to show you the process using the same install way and files from guide.
