        Use BEC to make restarts. 

        BEC/Config\Scheduler.xml

        My config for 4 hours restart: 

        <?xml version="1.0"?>
<Scheduler>

	
   	<job id='0'>
		<day>1,2,3,4,5,6,7</day>
		<start>040000</start> 
		<runtime>040000</runtime>
		<loop>-1</loop>
		<cmd>#shutdown</cmd>
	</job>	

</Scheduler>

        Alos i have infistar config to show restart messages:

        /* shows RESTART IN X MINS */
_USE_RESTART_TIMER = true;	/* true or false */
_RESTART_TIME_IN_M =   240;	/* restart time in minutes */
_SHOW_TIMER_IN_MIN =	[1,2,3,5,10,15,20,25,30,60];	/* minutes before restart, when message is shown */

        You need to make sure you have added your rcon password and external IP address to the EpochServer.ini

         

        [EpochServer]
BattlEyePath = C:\BattlEye
InstanceID = NA123
LogAbuse = 1
IP = 127.0.0.1  // <<< needs to be the external IP of your server
Port = 2302  // <<< game port
Password = changeme //  <<< needs to be set to your rcon password

        After you do this our force restart feature should work perfectly fine.
