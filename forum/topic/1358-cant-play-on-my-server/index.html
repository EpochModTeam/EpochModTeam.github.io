        Create a batch file in the root of your arma2/OA (combined) directory and add this:

        Expansion\beta\ARMA2OA.exe -client -mod=@DayZ_Epoch;Expansion\beta;Expansion\beta\Expansion -skipIntro -nosplash -showScriptErrors -noPause -world=Chernarus -name=<HC Name> -nosound -localhost=127.0.0.1 -connect=localhost -port=2302 -profiles=server -password=<server password>

        Replace anything in <..>  (pointy brackets) with your own details.. and run on same machine as your server.

         

        It does require a separate install of Arma OA and is supposed to require it's own unique (Steam) ID.
