        Hey dude, I play on the TTL epoch server and we have a team called the UN that help new survivors when we can. If you want to add me on your steam or come play on the server sometime I'll make you welcome. my steam name is wake_up_ne0 http://steamcommunity.com/profiles/76561198097258339

        Â 

        Firstly I would try and set up the game so it runs with a solid 60FPS, check in the towns and in busy areas. Adjust the view distance and graphics so your frame rate is solid.
        Also disabling mouse smoothing in controls help, make sure your talk button (capslock as default), is working and you know how to switch between channels such as direct communication to talk to people in the field. (< and > switch channels, you can switch to direct for speaking to randoms, vehicle chat if your in a car, global chat is EVERYONE-usually never talk in this one, only type.)

        Â 

        You should always try to find basics like bandages, painkillers and morphine especially when new try to have a few of each of these so you can recover.
        Zombies can only hit you when stationary (keep running if you have a horde on you). Also they can't get you in the big, dark green pine trees.
