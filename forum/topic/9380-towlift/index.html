        Ok, so you want to download this and put it into the main directory of "Mission.pbo".  

         https://dl.dropboxus...RTY_AND_LOG.zip

         

        Then open up "Init.sqf" and put this line of code in

         

        execVM "R3F_ARTY_AND_LOG\init.sqf"

         

        Now open up you "description.ext" and paste this line of code at the bottom

         

        #include "R3F_ARTY_AND_LOG\desc_include.h"
