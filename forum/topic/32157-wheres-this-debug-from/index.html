        Yes, just remove the keybind in theÂ dayz_spaceInterrupt.sqf file in Â dayz_code\actions\dayz_spaceInterrupt.sqf the lines you want to remove is on or near line 121

        if (_dikCode == 210) then {
		_nill = execvm "\z\addons\dayz_code\actions\playerstats.sqf";
};

        and on/near 142

        if (_dikCode in actionKeys "User20" && (diag_tickTime - dayz_lastCheckBit > 5)) then {
	dayz_lastCheckBit = diag_tickTime;
	_nill = execvm "\z\addons\dayz_code\actions\playerstats.sqf";
};

        and then copy/paste the file to your mission and add this to your compiles.sqf

        dayz_spaceInterrupt =  compile preprocessFileLineNumbers "custom\code\dayz_spaceInterrupt.sqf";

        If you use snap_building then just edit that file :) lines should be nearly the same
