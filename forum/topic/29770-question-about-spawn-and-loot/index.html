        Could be wrong but Arma 3 Epoch uses redis-server for the database not MySQL like A2 did.   In the main Arma 3 directory when you place the server files there is folder called DB in there is the redis-server.exe and config file etc. You would run that first then Epoch.

         

        Out of the box installation of the Server files, the two files  ( ..\Arma3Epoch\db\redis.config and ..\Arma3Epoch\EpochHive\EpochServer.ini) the passwords dont match so if you run with those Epoch works but you get what you are getting.   Spawning on island.  Run out gather some stuff.   Then logout and back in and you are a fresh spawn again.

         

        You will know its working when in the dos window of Redis server you should be getting messages about "## changes in ### seconds. " and "Background saving terminated with success" etc. every so often.
