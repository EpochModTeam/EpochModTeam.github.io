        Franzoart i was in the same boat as you wanted a custom server so was using bungles custom pack for a while then went onto Asian server pack. Once 1.2.1 came out i figured it was a great time to learn how to do it myself.

        Only basic addons like sarge ai, self bloodbaging, tent sleeping, sector FNG, its actual really easy to do.

        i would say try it your self start with something simple like the custom debug monitor then sarge ai the 2 easiest imo then work on something abit harder like sectors FNG or self bloodbaging.
