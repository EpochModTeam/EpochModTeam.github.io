        You would need to use to spawn any mas items. And I have no clue how do add them to the admin spawnlist.

        Custom loot spawn scripts have not been needed for in game loot since 0.3 came out.

        just edit loot.h file in your epoch server settings.pbo in the @epochhive folder.

        If you loaded the mods properly, you should already see mas vehicles and weapons in the admin menu.

        I'm not sure if the Epoch anti hack works the same as infi in that respect.

        Now if you think you aren't getting enough or quick enough loot, then yes you would want a loot spawn script but it's not great on performance once you get decent population
