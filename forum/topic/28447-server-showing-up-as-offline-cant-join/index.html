        The problem is DayZ Commander. Not your server. DayZ Commander uses a cached list of server that doesn't get updated frequently. Took 2 weeks for my servers to finally show up in DayZ Commander.

        Try DayZ Launcher instead. It directly queries the Steam master servers to get a live list of servers, instead of using a cached list that is god knows how old.
