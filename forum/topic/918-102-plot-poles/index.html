        Here's how to lookup all character id's that a specific PlayerUID has had. Replace __LOOKUP__VALUE__ with the character ID you want to look for...

        SELECT CharacterID from Character_DATA
WHERE PlayerUID = (
   SELECT PlayerUID
   FROM Character_DATA
   WHERE CharacterID = __LOOKUP__VALUE__)

        Enjoy
