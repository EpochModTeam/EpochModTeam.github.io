        Is the C130 added to the trader on your server to buy/sell ?

        If yes then you need to adjust the sell range, basically the plane isn't close enough to the trader to be sold, you can fix it by changing the values below in your init.sqf :

        dayz_sellDistance_air = 20; // 20 meters from trader, try 50
