        You need to set the battleye path in the Config.cfg in your BEC\Config folder, eg:

        BePath = F:\A3server\SC\BattlEye

        This needs to be the same as the bepath set in your server start command, eg:

        Â 

        start "arma3" "arma3server.exe" "-mod=@Epoch;@AllInArmaTerrainPack;@epochhive;" -bepath=F:\A3server\SC\BattlEye\ -port=2302 -maxMem=2048 -high -noPause -noSound -profiles=SC -name=SC "-config=SC\config.cfg" "-cfg=SC\basic.cfg"  -enableHT -autoinit
