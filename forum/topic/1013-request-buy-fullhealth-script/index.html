        Hey dude.

         

        I'm working on something like this for Epoch 1.0.2.2 using Lingor 1.5 - basically it's the same premise as adding custom traders but I'm adding 4 NPC's to be dotted around the map who you can mouse over, and get a heal option from. Will let you know how I go.

         

        All I have really is this so far, to be added to fnself_actions.sqf to bring up the menu. I'm not a coder though but slowly learning and this is all me hehe, so chances are that it probably won't work!

        if(cursorTarget isKindOf "unique doctor model" then {
    player addAction ["Get healed up","custom\heal.sqf",cursorTarget, 0, false, true, "",""];
    };

        I don't even know if that'd work to bring up a menu. But in theory that should bring up a menu option when you mouse over a particular model in game, just need to find a completely unique model.

         

        This calls a script in a custom folder in the mission.pbo that heals:

        r_player_blood = 11999;
r_player_inpain = false;
r_player_infected = false;
r_player_injured = false;
dayz_hunger	= 0;
dayz_thirst = 0;
dayz_temperatur = 100;
r_fracture_legs =	false;
r_fracture_arms =	false;
r_player_dead = false;
r_player_unconscious = 	true; // knock player out while healing to prevent possible pvp spam
r_player_loaded = 		false;
r_player_cardiac = 		false;
r_player_lowblood = 	false;
r_player_timeout =		20; // 20 second KO timer?
r_handlerCount = 		0;
disableUserInput false;
dayz_sourceBleeding =	objNull;
player setVariable ["USEC_injured",false,true];
{player setVariable[_x,false,true];} forEach USEC_woundHit;
player setVariable ["unconsciousTime", r_player_timeout, true]; //20 second KO
player setHit ['legs',0];
player setVariable ['hit_legs',0,false];
player setVariable['medForceUpdate',true,true];

distance = 5;

xyzaa = (getPosATL player) nearObjects ["Man", distance];
{

if (!(_x isKindOf "zZombie_Base") and !(_x isKindOf "Animal")) then { // this makes sure everything healed is a player.. .I think

_unit = _x;

usecBandage = [_unit];
publicVariable "usecBandage";
{_unit setVariable[_x,false,true];} forEach USEC_woundHit;
_unit setVariable ["USEC_injured",false,true];

_unit setVariable["USEC_lowBlood",false,true];
usecTransfuse = [_unit];
publicVariable "usecTransfuse";

_unit setVariable ["NORRN_unconscious", false, true];
_unit setVariable ["USEC_isCardiac",false,true];
_unit setVariable['medForceUpdate',true,true];

usecMorphine = [_unit];
publicVariable "usecMorphine";

"dynamicBlur" ppEffectAdjust [0]; "dynamicBlur" ppEffectCommit 5;

usecPainK = [_unit,player];
publicVariable "usecPainK";
_unit setVariable["medForceUpdate",true];

_text = format["You have been completely healed! This will only last till you logout though, so make sure to get the supplies you need while you still can!", xyzaa];
titleText [_text,"PLAIN DOWN"]; titleFadeOut 10;
};
} forEach xyzaa;

        You could jset set the blood value's global variable to the blood value total, and do a medforce update in probably 5-10 lines of code. As for broken legs, in pain, etc I'd have to look up the values. Also just disable user input instead of knocking him out, for the duration of the script if you dont want spam, and add a cool down timer before he can ask again. lastly make it cost something like gold, and remove it from their inventory then activate the script so it doesnt get abused with free heals.

         

        A quick google search will reveal this type of code exists in many files on armaholics, for medic actions and such in different multiplayer missions which can help guide you on how to make one in dayz (just decompile the dayz code and look at medical scripts). Lastly many dayz "anti-hacks" which are just really re-purposed hacks, already have the type of script you are looking for, it would just need to be re-purposed for your doctor NPC and some self actions. So instead of reinventing the wheel just take something open source that works and edit to your liking while crediting original authors in the code via a comment. Hope my suggestions help your thought process!
