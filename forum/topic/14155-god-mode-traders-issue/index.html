        Start vanilla Arma 2 and go to the 2D editor, make sure it's not 3D editor (Single player>Editor>Chernarus) place a unit wherever you want it to be, save the mission then go to "C:\Users\YOURUSERNAME\Documents\ArmA 2 Other Profiles\[OSC]%20Kar\missions\whateveryounamedyourmission.chernarus" (Or just hit start>documents>arma2otherprofiles>name>mission). Open Mission.sqm, find your unit, should look something like this 

        	class Groups
	{
		items=1;
		class Item0
		{
			side="WEST";
			class Vehicles
			{
				items=1;
				class Item0
				{
					position[]={5067.8472,338.99286,9727.7734};
					id=0;
					side="WEST";
					vehicle="USMC_Soldier";
					player="PLAYER COMMANDER";
					leader=1;
					skill=0.60000002;
				};
			};
		};
	};

        then just copy down the position on that line. The reason why your worldspace didn't work was because it pulled the 3D worldspace instead of the 2D. I went in the 3D editor and put your worldspace in and it showed it was SE part of the tarmac so I just went to 2D editor placed it there it showed in 3D and gave you the new position
