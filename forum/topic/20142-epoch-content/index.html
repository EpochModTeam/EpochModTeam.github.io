        Cheers

         

        The default epoch doesnt have that much new features, more like small isues fixed and looks enhanced. All attention is going to Arma 3 Epoch now.

         

        Epoch is supported on ALOT of maps ^^

         

        However there are quite the amazing communti scripts/extensions now which can give your playtime some new flavour!

         

        Examples:

         

        - Snap Building: Build assist so walls and stuff can connect nicely for you without cursing

        - Vector Building: U can turn builable in anyways possible!

        - Single Currency: No more gold/silver/briecase/copper/tin items, use 1 currency: Gold coins, (not an inventory item eather)

        - Plot 4 Life: never replace a plot anymore when u died

        - Plot Management: add friends to plot so they can build always, even when u are ofline.

        - Various Craft mods : that allow buimding of alot of custom buildings/objects

         

        Overpoch is stable.

         

        It depends from server to server however if the coders are capable of twisting it in the right direction ( there are sill alot of servers without a technical scripter behind it).

         

        Overwatch + Epoch is Epoch with extra guns and vehicles ( and skins, if added correctly).

         

        You even have Overpochins now or Pochins: (Overwatch) + Epoch + Origins.
