        23:19:53 Server: Object 48:1305 not found (message 94)
23:19:53 Server: Object 48:1307 not found (message 70)

        These refer to networked objects that are non-existent. Ex. if an object is deleted and referenced after its deletion it will throw this warning (gathered from some snooping in logs and such). This is completely normal, don't worry about these.
