        DZMS and WAi work fine wih 1.0.5.1 no issue so far.

         

        Installing like said on addon install in this forum.

         

        Download both DZMS and WAI master folder.

         

        Move into your server.pbo DZMS and WAI folder

        open your server.pbo/system/server_monitor.sqf and look at the end of the file you will see:

        		diag_log format["Total Number of spawn locations %1", actualSpawnMarkerCount];
		
		endLoadingScreen;
	};

	allowConnection = true;	
	sm_done = true;
	publicVariable "sm_done";
};
 

        Add right before " allowConnection = true; "

         
        [] ExecVM "\z\addons\dayz_server\DZMS\DZMSInit.sqf";
        [] ExecVM "\z\addons\dayz_server\WAI\init.sqf";
         
        it must look like this:
         
        		diag_log format["Total Number of spawn locations %1", actualSpawnMarkerCount];
		
		endLoadingScreen;
	};

	[] ExecVM "\z\addons\dayz_server\DZMS\DZMSInit.sqf";
	[] ExecVM "\z\addons\dayz_server\WAI\init.sqf";
	allowConnection = true;	
	sm_done = true;
	publicVariable "sm_done";
};

        Save, repack update your server and that's done.

         

        PS: You may have to configure DZMS and WAI ( look at the readme to know how)
