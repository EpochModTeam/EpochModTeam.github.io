        Hello,

        I show you a part of my scheduler.xml with comments for you, it may help:

        <job id="2">
<time>05:59:55</time>        <!-- 5sec before shutdown -->
<delay>000000</delay>
<day>1,2,3,4,5,6,7</day>
<loop>0</loop>
<cmd>c:\epoch server 1.0.4.2\run.bat</cmd>    <!-- call your server run.bat -->
<cmdtype>1</cmdtype>                <!-- 1 = external command -->
</job>

<job id="3">
<time>06:00:00</time>    <!-- run at 6'o Clock in the morning -->
<delay>000000</delay>
<day>1,2,3,4,5,6,7</day>
<loop>0</loop>        <!-- a loop isnt needed for restart -->
<cmd>#shutdown</cmd> <!-- shutdown the server -->
<cmdtype>0</cmdtype>

        after you'll have to set a delay into the run.bat, and include bec launching aswell. Here is my file:

        timeout 20

@echo off
set armapath="C:\EPOCH SERVER 1.0.4.2"
cd %armapath%
start "arma2" /min "Expansion\beta\arma2oaserver.exe" -port=2302 "-config=instance_11_Chernarus\config.cfg" "-cfg=instance_11_Chernarus\basic.cfg" "-profiles=instance_11_Chernarus" -name=instance_11_Chernarus "-mod=@DayZ_Epoch;@DayZ_Epoch_Server;"

timeout 20

echo
set becpath="C:\EPOCH SERVER 1.0.4.2\instance_11_Chernarus\Bec"
cd %becpath%
start "" "bec.exe" -f config.cfg
echo.
echo Server Started 100%
cls
@exit

        I tried it on 2 servers, and it work well. Hope this will work for you too
