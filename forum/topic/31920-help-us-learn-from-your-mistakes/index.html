        We run a bunch of scripts on our A2 Epoch servers with no troubles.

         

        Anit-hack: the best one is active admins.

        Second-best is to run BE with the latest filters.

        The big gain with a purchased admin tool is that BE filters are provided for most popular addons. If you can tackle that bit then see the above.

        Having an admin tool that permits you to clone keys, unlock vehicles, spawn vehicles and read safe and lock box combos is handy

        A good object and delete tool is helpful

        There are some good admin tools around for free which you can further modify to suite your needs.

         

        Deployable bike scripts differ primarily in what building materials are needed (ours uses a toolbox) and what you get back when you deconstruct it. The same goes for SelfBB. Epoch provides a selfBB now so the addons have less role these days.

         

        The rest is really up to you and you can configure any of them to change these parameters.

         

        The changekey script as developed by otternas and maintained by others is popular and works pretty well.

        if you cater to people who build alot then snap build pro and vector build are worth the time, and are available as pre-prepared packages you basically plug in. For a server for builders,  I would look into right click buildables.

         

        If you wish to use lift/tow check the A3 forums for links to some newer scripts with nicer interfaces and modernized code.

         

        Keep an eye on the number of objects on the server. Keep the base destruction time low enough that < 5-6K objects are on the server at any time.
