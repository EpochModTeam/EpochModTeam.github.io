        Firstly I opened up my Mission pbo, and created these files:

        Mission Pbo > Custom > loadouts

        After creating those files, I created a .sqf file called: init.sqf, and placed it in the loadouts file.

        I then used this code within it:

        ~~~// Admin Loadouts //~~~
{
// adminone
if ((getPlayerUID player) in ["ADMINONEUID"]) then {
	[] execVM "custom\loadouts\adminoneloadout.sqf";

        - adminoneloadout.sqf

        DefaultMagazines = ["ItemBandage","8Rnd_9x18_Makarov","ItemPainkiller","ItemSodaCoke","FoodSteakCooked"];

        if (!isDedicated) then {

        But when I respawn, none of the load outs work. I cannot tell what the error is...

        Okay, nevermind. I've fixed what I needed doing.

        This is what I've done.

        Find this in the init.sqf:

        if (!isDedicated) then {

        Add this below it:

        	[] execVM "custom\loadouts\init.sqf";

        Create the file directory of:

        Custom > Loadouts

        Create init.sqf in Loadouts and use this as a template (paste in):
         

        //Default Loadout
DefaultMagazines = [""];
DefaultWeapons = [""];
DefaultBackpack = "";
DefaultBackpackWeapon = "";

//Admin One
if ((getPlayerUID player) in ["Admin One"]) then {
	DefaultMagazines = [""];
	DefaultWeapons = [""];
	DefaultBackpack = "";
	DefaultBackpackWeapon = "";
	};

//Admin Two
if ((getPlayerUID player) in ["Admin Two"]) then {
	DefaultMagazines = [""];
	DefaultWeapons = [""];
	DefaultBackpack = "";
	DefaultBackpackWeapon = "";
	};

//Donator One
if ((getPlayerUID player) in ["Donator One"]) then {
	DefaultMagazines = [""];
	DefaultWeapons = [""];
	DefaultBackpack = "";
	DefaultBackpackWeapon = "";
	};

//Donator Two
if ((getPlayerUID player) in ["Donator Two"]) then {
	DefaultMagazines = [""];
	DefaultWeapons = [""];
	DefaultBackpack = "";
	DefaultBackpackWeapon = "";
	};

        Once added the ID's in the right section, repack, upload. Your good to go.
