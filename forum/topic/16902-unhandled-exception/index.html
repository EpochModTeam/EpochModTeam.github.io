        This happens to me too, but I work around playing Arma 3 Epoch mod with a Windows batch script:

        set ARMA3_PATH=D:\Program Files (x86)\Steam\SteamApps\common\Arma 3 
set DEFAULT_PARAMETERS=-nosplash -skipintro -world=empty -noFilePatching
cd /D "%ARMA3_PATH%"
arma3.exe %DEFAULT_PARAMETERS% %DEVELOPMENT_PARAMTERS% "-mod=@Epoch"

        Notice how my file path is 'D:\', and not the default of the root 'C:\' drive for my installation.  It may have to do with that, or something else entirely.

         

        If I had source code access, I could easily fix it.
