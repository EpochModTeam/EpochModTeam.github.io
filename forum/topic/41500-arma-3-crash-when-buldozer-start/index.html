        Try to run the "DevP.bat" in the "Arma 3 Tools" folder as it will update bulldozer.exe and ensure everything else is setup.

        Also in Object Builder settings check these are set:

        External Viewer: 
        P:\buldozer.exe -buldozer -name=Buldozer -window -noLand -exThreads=0 -noLogs -noAsserts -cfg=p:\buldozer.cfg 

        Path for textures:
        P:\
