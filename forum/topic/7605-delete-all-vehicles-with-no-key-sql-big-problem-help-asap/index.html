        //Show normal vehicles that epoch spawns on restart

        SELECT * FROM object_data, allowedvehicles WHERE allowedvehicles.className = object_data.Classname AND
		   	object_data.CharacterID = '0'

        //Show all vehicles in the database

        SELECT * FROM object_data, allowedvehicles WHERE allowedvehicles.className = object_data.Classname

        If you know the exact date of when the problem started do this (put the date there):

        SELECT * FROM object_data, allowedvehicles WHERE allowedvehicles.className = object_data.Classname AND
`Datestamp` >= '2014-02-15 16:50:20'

        After you make your selections, either delete them with the checkboxes or replace SELECT * FROM with DELETE FROM to delete them.

         

        I've seen tons of SQL events and stuff....but honestly this is the best way to get proper data....the sql you posted might delete modular building as well. If you havent spawned any custom vehicles on the map then this will work 100%.

         

        And after that its very easy to maintain...any new type of vehicle you add, just add it to your allowedvehicles table so you always get proper results.

        Sometimes re-inventing the wheel is not good :) Good old allowedvehicles existed in the past....cant think of a reason why they removed it.

         

        ps: to see all the 'bought' cars that players got from vendors:

        SELECT * FROM object_data, allowedvehicles WHERE allowedvehicles.className = object_data.Classname AND
		   	object_data.CharacterID != '0'
