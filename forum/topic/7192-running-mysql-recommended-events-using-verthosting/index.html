        Run sql query under your epoch database e.g.

        CREATE EVENT removeDamagedVehicles
    ON SCHEDULE EVERY 1 DAY
    COMMENT 'Removes damaged vehicles'
    DO
    DELETE FROM `object_data` WHERE Damage = 1;
