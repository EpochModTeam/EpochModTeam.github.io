        I fixed my teleport issues by adding PlayerID exceptions to the antihack.sqf. Essentally, if you're an admin, you're exempt from the file.

         

        Just under this:

        waitUntil {vehicle player == player};

        Add this:

        if (!(getPlayerUID player) in ["XXXXXXX","XXXXXXX"]) then {

        where the Xs are your admins' UIDs.

         

        Then at the bottom of the whole thing, add an extra

        };
