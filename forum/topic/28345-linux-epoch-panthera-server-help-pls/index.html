        All you need to do is use the dayz_epoch_16.panthera2.pbo  and in the cfgdayz/server.cfg file change

        class Missions
{
	class DayZ {
		template = dayz_epoch_11.chernarus;
		difficulty = "veteran";
	};
};

        to 

        class Missions
{
	class DayZ {
		template = dayz_epoch_16.panthera2;
		difficulty = "veteran";
	};
};
