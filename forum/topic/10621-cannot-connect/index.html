        It's because your arma 2 install was updated to 1.63. You need to downgrade to 1.62 to play on epoch servers atm. I'll try and explain it as best as I can for you if you are using the steam version.

         

        1. Make sure Dayzcommander is closed.

        2. Open up Steam

        3. Right click Arma 2 OA

        4. Go to properties.

        5. Search for Verify game cache

        6. When the verification finishes start Arma 2 OA

        7. When it gets to the player profile screen exit Arma 2 OA

        8. Open Dayzcommander

        9. Click Install/update

        10. Install the beta patch where it says Arma 2.

        11. It will say wrong Cdkey don't worry its just a bug

        12. You will now be able to join servers.

        13. If you have problems PM me and i can remotely connect to solve your problem.
