        It's easier (IMO) to find errors and such if you get rid of all the extraneous comments like I've done here to your first few lines:

         

        [[16328.35, 18390.408,0.001],4,1,"Random",4,"","Bandit2_DZ","Random"] call spawn_group;
         
        [[4646.3916, 16736.672,0.001],4,1,"Random",4,"","","Random"] call spawn_group;
         
        [[4693.1245, 16756.039,0.001],4,1,"Random",4,"","","Random"] call spawn_group;
         
        I've also found that these scripts can be really sensitive to formatting...so in your position line where there is an extra space, delete it e.g.
        [[16328.35, 18390.408,0.001]   should be [[16328.35,18390.408,0]   (you also don't need 0.001, just a 0 will work fine).
         
        For the paradrops, I had better luck using a start position instead of 0,0,0
         
        I don't spot any major errors in your script.  I found that mine tended to work nearly flawlessly when I got rid of ALL the comments though.
