        Make sure on the launcher you clicked "install". To verify the mod is installed go to Steam, right click on Arma 3, click properties, select the local files tab, and click browse local files. In the folder that gets opened up make sure there is a file called "@Epoch". To make sure the mod is enabled, when you launch Arma 3 go to the main menu, click configure, click expansions and make sure Epoch Mod is enabled and has a check mark next to it.
