        as far as it not starting.. do you mean it doesnt initialize? or you just get kick messages from trying to access it?

        Not sure how long its been since you have used it, if you havent added in the new ID given after the gamespy shutdown it wont allow you access to it. 

        As far as the emails, just email him from the email you used when purchasing it and request the updated version.

         

        If its not initializing then you have an error in your   dayz_server\init\server_functions.sqf  

         

        If both files from the anti hack are located in dayz_server\init and this line:

         

        [] spawn {[] execVM "\z\addons\dayz_server\init\AH.sqf";};

         

        is at the top of your dayz_server\init\server_functions.sqf it should work fine.

         

        The readme should have the instructions to enable teleporting.
