        got the PM, I can whip up a quick config.

        It will have all the Random/Dynamic/Static and Land/Air Vehicle turned off as well.

        It will have the a3eai_custom_defs turned to true.

        This is your A3EAI_config.sqf:

        /*

        A3EAI Configuration File

        Description: Contains all configurable settings of A3EAI. Contains settings for debugging, customization of AI units, spawning, and loot.
