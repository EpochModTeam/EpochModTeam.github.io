        Nothing is removed based on if you die or not. If anything the building will disappear 6 days from when you last placed them including plot poles. This can be changed server side to anything you want if it is set to 0 then all objects will disappear on restart.

         

        Hiveext.ini

        ;Negative values will disable this feature
        ;0 means that ALL empty placed items will be deleted every server restart
        ;A positive number is how old (in days) a placed empty item must be, in order for it to be deleted
        CleanupPlacedAfterDays = 14
