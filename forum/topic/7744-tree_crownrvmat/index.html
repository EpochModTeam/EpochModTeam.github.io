        Google Panthera map v2.9, then go to the Armaholic link for Icebreaker's version 2.9 of Panthera. At the bottom of the page are two ftp links; download the 7zip file and unpack to your DayZ game folder (the one with all the @dayzxxx folders).

        It should unpack as @Panthera sub-folder. Then, in Commander settings - Additional Launch Parameters - put this EXACT phraseÂ 
        -mod=@Dayz_Epoch;@Panthera Â  Â  (<<< that last word being the name of the folder you just unpacked/installed).
