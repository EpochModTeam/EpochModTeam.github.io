        The config file of R3F logistics lists what objects get these scroll options, just empty these lists and the load in and take object options are gone.

         

        the lists you need to modify are:

        - R3F_LOG_CFG_transporteurs for the "load in" option.

        - R3F_LOG_CFG_objets_transportables for the "take this object" option.
