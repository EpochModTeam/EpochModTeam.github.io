        Hello,

        please try a clean reinstall of Arma 2 CO:

        1. Go to your Steam Library and uninstall Arma 2, Arma 2 Operation Arrowhead
        and all DLCs (if applicable). To do this, simply right-click on the game you
        wish to uninstall and click on Delete Local Content.

        2. Delete "<steam>/steamapps/common/arma 2" and "<steam>/steamapps/common/arma
        2 operation arrowhead" folders and Arma 2's shortcuts on your desktop (if any).
        Delete "Documents/ArmA 2" and, if present, "Documents/ArmA 2 OA". Then reboot
        your computer.

        3. Restart Steam, make sure it is being ran as Administrator (right-click on
        the Steam icon and choose "Run As Administrator").

        4. Install Arma 2.

        5. Once Arma 2 is installed, run it once, then close it.

        6. Install Arma 2 Operation Arrowhead.

        7. Run Arma 2 OA once. If you purchased Arma 2 ACR DLC, it will be installed
        now (you may have to restart Steam and try again if it is not).

        8. Close Arma 2 OA.

        9. Install any other DLCs you have (BAF, PMC) and run each one once.

        10. Launch Arma 2 OA once more and make sure all DLCs are correctly installed
        and visible in the main menu.

        11. You now have fully functioning Arma 2 Combined Operations. If you wish to
        install DayZ mod, please continue to step 12.

        12. Right-click on Arma 2 OA, click on Properties, then go to Beta tab.

        13. Opt in to the Beta program by selecting Beta.

        14. Wait until OA is patched.

        15. Install Arma 2 DayZ mod via Steam (if you cannot see it in your Library,
        please go to Store tab and search for Arma 2 DayZ mod and add it to your
        Library).

        16. Play Arma 2 DayZ mod.

        Please note that some servers might be yellow or red and thus inaccessible
        depending on your and their game version. If you need an older Beta version of
        Arma 2 OA in order to play on your favorite server, please close DayZ, and opt
        into an older Beta patch in Arma 2 OA Steam properties.

        Also please remember that mods are community created content, not a part of the
        game itself. In case there is any problem with modded game installation, please
        contact directly the particular mod author or community.
