        If your Server exefile has been renamed from arma2oaserver.exe to something else. you need to specify this in the Bec config file in the Misc section.
        ServerExeName = somename.exe
        if that dosent work. you can try starting up Bec with the --dsc option. like Bec.exe -f myconfig.cfg --dsc
