        You need to edit your HiveExt.ini file with the following:

         

        ;Negative values will disable this feature

        ;0 means that ALL empty placed items will be deleted every server restart
        ;A positive number is how old (in days) a placed empty item must be, in order for it to be deleted
        CleanupPlacedAfterDays = -1

        Ok there is a line in your hiveExt.ini i got this from no one i had to find it from my friend. 

         

         

        ;Which table should the objects be stored and fetched from ?

        Table = object_data

        Make sure that where it says object_data that it is case sensitive then it should stop cleaning objects.
