        **SOLVED**

        Solution for others who are stuck:

        The issue is currently a issue with on Battleye's side! They developed the newest version on GLIBC 2.15 I was running GLIBC 2.13 and I believe most servers are right now. To fix that issue please follow this link here: http://stackoverflow.com/questions/10863613/how-to-upgrade-glibc-from-version-2-13-to-2-15-on-debian This will guide you through updating that and it should work like a charm!
