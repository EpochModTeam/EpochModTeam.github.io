        There should be an option where AIs won't have gear/weapon when you overrun them... try to set this option to false... wait, i'll check this...

        Â 

        Â 

        In DZMS at about line 29 set this to false

        // Do you want AI that players run over to not have gear?
// (If DZMSCleanDeath is true, this doesn't matter)
DZMSRunGear = false;

        Don't know if DZAI has something similar... but WAI don't have this option afaik... so check it out in DZMS and hopefully it will work then
