        Yes. The IP as actually not needed in the CMD. If you put localhost in there (127.0.0.1) it doesn't run on the NIC which is connected to the network :)

        FYI : If you only need local IP, using 127.0.0.1 is no problem. For example if you want your Arma server to connect to Redis (which runs local) you have no problems using localhost. But the game itself should run on *all* attached NICs.
