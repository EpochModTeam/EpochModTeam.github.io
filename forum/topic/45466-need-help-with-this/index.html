        *SOLVED*

        Remove: 

        cutText ["ROUND STARTS IN...", "PLAIN DOWN"];

        Then replace with:

        sleep 10;

        ["<t size='0.60' font='Zeppelin32' color='#B22222' shadow='2' align='center'>Add text here</t><br />",0,1,3,0.9] spawn BIS_fnc_dynamicText;//Change color code and text
