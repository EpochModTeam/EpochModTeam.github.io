        Drop Blue Phoenix and get infistar working. It is not very hard. Explain what your problem is with infistar and I bet many people will tell you your problem as infistar is 4-5 easy steps to install and it does work for hundreds of server owners.

         

        Add this AT THE VERY TOP OF server_functions.sqf ABOVE EVERYTHING ELSE

        [] spawn {[] execVM "\z\addons\dayz_server\init\AH.sqf";};

        Place the ah.sqf and ahconfig.sqf inside your dayz_server -> init -> here. (same location like "server_functions.sqf")

         

        Remove or comment out these two lines inside your mpmission init.sqf (to comment them out just put two // infront of them)

        [] execVM "\z\addons\dayz_code\system\antihack.sqf";

#include "\z\addons\dayz_code\system\REsec.sqf"

        Check if you got this inside your mpmission init.sqf. If you do set it like this. They both have to be = 0;

        dayz_antihack = 0; // DayZ Antihack / 1 = enabled // 0 = disabled
dayz_REsec = 0; // DayZ RE Security / 1 = enabled // 0 = disabled

        In the "AHconfig.sqf" you will find. You have to add your PlayerID (NOT GUID) in there to become an admin.

        /*  SUPER ADMIN HERE      */ _SuperLevel_List = ["0","0","0"]; //do not have a , at the end.

        Update/replace your batteye filters with the ones infistar provides you with.

         

        Done.
