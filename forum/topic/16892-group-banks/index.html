        I had an idea for a group banking system, for example they could open up a dialog when at a bank and add / remove members to the group bank, then each time someone deposits / withdraws they get an option would they like to do it from personal or group.

         

        This way players can work together and if they want to can have a join fund even when there fellow group members are offline.

         

        Was just curious to see if any one else has done this, before I start looking in to it any further.
