        you need to go to the data base and run these SQL

        please make a backup just in case.

         

        if your object data base isnt spelled like this  Object_DATA change it before you run this

        UPDATE Object_DATA SET Classname = 'ArmoredSUV_PMC_DZ' WHERE Classname = 'ArmoredSUV_PMC_DZE'; 
UPDATE Object_DATA SET Classname = 'CH_47F_EP1_DZ' WHERE Classname = 'CH_47F_EP1_DZE'; 
UPDATE Object_DATA SET Classname = 'GAZ_Vodnik_DZ' WHERE Classname = 'GAZ_Vodnik_DZE'; 
UPDATE Object_DATA SET Classname = 'HMMWV_M1151_M2_CZ_DES_EP1_DZ' WHERE Classname = 'HMMWV_M1151_M2_CZ_DES_EP1_DZE'; 
UPDATE Object_DATA SET Classname = 'HMMWV_M998A2_SOV_DES_EP1_DZ' WHERE Classname = 'HMMWV_M998A2_SOV_DES_EP1_DZE'; 
UPDATE Object_DATA SET Classname = 'LandRover_MG_TK_EP1_DZ' WHERE Classname = 'LandRover_MG_TK_EP1_DZE'; 
UPDATE Object_DATA SET Classname = 'LandRover_Special_CZ_EP1_DZ' WHERE Classname = 'LandRover_Special_CZ_EP1_DZE'; 
UPDATE Object_DATA SET Classname = 'Mi17_DZ' WHERE Classname = 'Mi17_DZE'; 
UPDATE Object_DATA SET Classname = 'Offroad_DSHKM_Gue_DZ' WHERE Classname = 'Offroad_DSHKM_Gue_DZE'; 
UPDATE Object_DATA SET Classname = 'Pickup_PK_GUE_DZ' WHERE Classname = 'Pickup_PK_GUE_DZE'; 
UPDATE Object_DATA SET Classname = 'Pickup_PK_INS_DZ' WHERE Classname = 'Pickup_PK_INS_DZE'; 
UPDATE Object_DATA SET Classname = 'Pickup_PK_TK_GUE_EP1_DZ' WHERE Classname = 'Pickup_PK_TK_GUE_EP1_DZE'; 
UPDATE Object_DATA SET Classname = 'UAZ_MG_TK_EP1_DZ' WHERE Classname = 'UAZ_MG_TK_EP1_DZE'; 
UPDATE Object_DATA SET Classname = 'UH1H_DZ' WHERE Classname = 'UH1H_DZE'; 
UPDATE Object_DATA SET Classname = 'UH1Y_DZ' WHERE Classname = 'UH1Y_DZE'; 
UPDATE Object_DATA SET Classname = 'UH60M_EP1_DZ' WHERE Classname = 'UH60M_EP1_DZE';

        and if your Traders data class name is not Traders_DATA you will have to change it before you run this

        UPDATE Traders_DATA SET item = 'ArmoredSUV_PMC_DZ' WHERE item = 'ArmoredSUV_PMC_DZE'; 
UPDATE Traders_DATA SET item = 'CH_47F_EP1_DZ' WHERE item = 'CH_47F_EP1_DZE'; 
UPDATE Traders_DATA SET item = 'GAZ_Vodnik_DZ' WHERE item = 'GAZ_Vodnik_DZE'; 
UPDATE Traders_DATA SET item = 'HMMWV_M1151_M2_CZ_DES_EP1_DZ' WHERE item = 'HMMWV_M1151_M2_CZ_DES_EP1_DZE'; 
UPDATE Traders_DATA SET item = 'HMMWV_M998A2_SOV_DES_EP1_DZ' WHERE item = 'HMMWV_M998A2_SOV_DES_EP1_DZE'; 
UPDATE Traders_DATA SET item = 'LandRover_MG_TK_EP1_DZ' WHERE item = 'LandRover_MG_TK_EP1_DZE'; 
UPDATE Traders_DATA SET item = 'LandRover_Special_CZ_EP1_DZ' WHERE item = 'LandRover_Special_CZ_EP1_DZE'; 
UPDATE Traders_DATA SET item = 'Mi17_DZ' WHERE item = 'Mi17_DZE'; 
UPDATE Traders_DATA SET item = 'Offroad_DSHKM_Gue_DZ' WHERE item = 'Offroad_DSHKM_Gue_DZE'; 
UPDATE Traders_DATA SET item = 'Pickup_PK_GUE_DZ' WHERE item = 'Pickup_PK_GUE_DZE'; 
UPDATE Traders_DATA SET item = 'Pickup_PK_INS_DZ' WHERE item = 'Pickup_PK_INS_DZE'; 
UPDATE Traders_DATA SET item = 'Pickup_PK_TK_GUE_EP1_DZ' WHERE item = 'Pickup_PK_TK_GUE_EP1_DZE'; 
UPDATE Traders_DATA SET item = 'UAZ_MG_TK_EP1_DZ' WHERE item = 'UAZ_MG_TK_EP1_DZE'; 
UPDATE Traders_DATA SET item = 'UH1H_DZ' WHERE item = 'UH1H_DZE'; 
UPDATE Traders_DATA SET item = 'UH1Y_DZ' WHERE item = 'UH1Y_DZE'; 
UPDATE Traders_DATA SET item = 'UH60M_EP1_DZ' WHERE item = 'UH60M_EP1_DZE';

        finally to spawn vehicles with full ammo. you need to edit your dynamic_vehicle.sqf and remove the E from these vehicles:

        ArmoredSUV_PMC_DZE
        CH_47F_EP1_DZE
        GAZ_Vodnik_DZE
        HMMWV_M1151_M2_CZ_DES_EP1_DZE
        HMMWV_M998A2_SOV_DES_EP1_DZE
        LandRover_MG_TK_EP1_DZE
        LandRover_Special_CZ_EP1_DZE
        Mi17_DZE
        Offroad_DSHKM_Gue_DZE
        Pickup_PK_GUE_DZE
        Pickup_PK_INS_DZE
        Pickup_PK_TK_GUE_EP1_DZE
        UAZ_MG_TK_EP1_DZE
        UH1H_DZE
        UH1Y_DZE
        UH60M_EP1_DZE
