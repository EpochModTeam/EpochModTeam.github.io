        If you haven't figured out how to fix it yet,

        What that error means is that the ID number in the database has been duplicated.

        So at the end of it, you will see the chem lights, and then the cinder blocks you added in.

        The problem is you have made the cinder blocks and the chemlights the same ID, so when the database is called for information by the server, it won't know which item to load into the traders.

        You must change either one of those items to a number that isn't used, any number, as long as it's not being used by another item.

        Eg. Change cinder blocks 6741 to 6742, or even 9532.
