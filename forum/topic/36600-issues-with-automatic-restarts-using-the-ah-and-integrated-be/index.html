        Make sure your EpochServer.ini has the external IP address instead of 127.0.0.1 and your current rcon password instead of "changeme".

        [EpochServer]
BattlEyePath = C:\BattlEye
InstanceID = NA123
LogAbuse = 1


IP = 127.0.0.1
Port = 2302
Password = changeme
...

        Failing that check your EpochServer.log file for more information.
